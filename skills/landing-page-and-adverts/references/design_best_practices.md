# Landing Page Design & Eye-Tracking Best Practices

This reference outlines the visual and structural elements that maximize conversion based on eye-tracking studies and UX research.

## 1. Eye-Tracking Patterns
- **F-Pattern**: Users scan the top horizontally, then move down and scan horizontally again, then move down the left side. Place the most critical information (Logo, Headline, CTA) along this path.
- **Z-Pattern**: Common for pages with less text. The eye moves from top-left to top-right, then diagonally to bottom-left, then across to bottom-right. Ideal for simple landing pages.
- **Directional Cues**: Use arrows or the gaze of people in images to point toward the CTA. Humans naturally follow the direction of sight or pointers.
- **The Fold**: While users do scroll, the "Above the Fold" area still gets the most attention. Ensure the Headline, Subheadline, and primary CTA are visible without scrolling.

## 2. Structural Elements (The Anatomy)
| Element | Purpose | Best Practice |
| :--- | :--- | :--- |
| **Hero Shot** | Visual context | Show the product in use or the result of using it. |
| **Headline** | Grab attention | Focus on the #1 benefit, not a feature. |
| **Subheadline** | Explain value | Clarify how the benefit is delivered. |
| **Social Proof** | Build trust | Use real photos with testimonials and specific metrics. |
| **Features/Benefits** | Logical support | Use bullet points. Benefits should lead (What's in it for them?). |
| **Primary CTA** | Drive action | Use a high-contrast color and action-oriented text. |
| **Frictionless Form** | Lead capture | Keep fields to a minimum (3-5 max for initial lead gen). |

## 3. Visual Hierarchy & UX
- **Contrast**: Use color contrast to make the CTA button the most prominent element on the page.
- **White Space**: Prevent cognitive overload by giving elements room to breathe.
- **Typography**: Use large, legible headings and clear body text (minimum 16px).
- **Mobile First**: Ensure the page is fully responsive. 50%+ of traffic is likely mobile.

## 4. Technical Performance
- **Page Speed**: Every 1-second delay in load time can decrease conversions by 7%. Compress images and minimize heavy scripts.
- **AEO (Answer Engine Optimization)**: Structure content with clear headings (H1, H2) and FAQs to be easily parsed by AI search engines (Google Gemini, Bing Copilot).
